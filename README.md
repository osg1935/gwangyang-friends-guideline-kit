# 광양프렌즈 Guideline Kit HTML

세 캐릭터 백운이, 기정이, 매향이를 종합한 광양프렌즈 캐릭터 가이드라인 HTML입니다.

구성:

- `광양프렌즈 캐릭터 가이드라인.html`: 종합 가이드라인 본문
- `assets/css/style.css`: 종합본 전용 스타일
- `assets/광양프렌즈 캐릭터 사이즈.svg`: 광양프렌즈 캐릭터 사이즈 기준 이미지
- `assets/baeguni`: 백운이 기존 가이드라인 에셋
- `assets/maehyangi`: 매향이 기존 가이드라인 에셋
- `assets/gijeongi`: 기정이 기존 가이드라인 에셋

화면/출력:

- 각 시트는 A4 가로 비율의 독립 페이지로 구성됩니다.
- 앞부분은 표지, 캐릭터 설정, 사이즈 기준으로 구성됩니다.
- 사이즈 기준 페이지에는 기본 사이즈 기준과 6개 응용동작 사이즈 적용 예시가 함께 표시됩니다.
- 각 캐릭터 소개 페이지의 브랜드 컬러는 HEX, RGB, CMYK 값을 함께 표시합니다.
- 이후에는 백운이, 기정이, 매향이 순서로 개별 가이드라인 페이지가 이어집니다.
- 각 캐릭터 소개 페이지 안에 브랜드 컬러를 함께 포함해 전체 장수를 줄였습니다.
- 각 캐릭터별 동작가이드 10종, 3D 3종, 굿즈 전체 항목을 포함합니다.
- 상단 내비게이션으로 장별 이동이 가능합니다.
- 이미지 영역은 클릭하면 크게 볼 수 있습니다.
- 인쇄 시 `@page { size: A4 landscape; margin: 0; }` 기준으로 페이지가 나뉩니다.
